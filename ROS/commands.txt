Название: strace
Описание: команда strace показывает все системные вызовы программы, которые та отправляет к системе во время выполнения, а также их параметры и результат выполнения.
Синтаксис: strace [опции] [команды] [аргументы]
Пример: htop ls
Примечание: С помощью опции -с вы можете собрать статистику для системных вызовов, которые использует программа
===
Название: tree
Описание: она умеет изображать структуру всех каталогов, имеющихся на компьютере, в виде дерева. 
Синтаксис: tree [опции]
Пример: tree 
Примечание: Чтобы сократить кол-во демонстрирующихся объектов (показать только папки) - $ tree -d
===
Название: journalctl
Описание: Используется для просмотра логов определенного сервиса или загрузки
Синтаксис: journalctl [опции]
Пример: journalctl -u [service-name]
Примечание: Если хотим посмотреть только сообщения ядра используйте опцию -k
===
Название: diff
Описание: Создает список изменений, котоые необходимо внести в первый файл, чтобы он соответствовал второму файлу
Синтаксис: diff [файл1] [файл2]
Пример: diff file1.txt file2.txt
Примечание: Для более яркого отображения можно установить colordiff
===
Название: time
Описание: Показывает время выполнения команды
Синтаксис: time [опции] [команда]
Пример: time ls

Название: asciinema
Описание: Записывает ваши действия в терминале
Синтаксис: asciinema rec [имя-файла] | asciinema play [имя-записи]
Пример: asciinema rec test
Примечание: Запись можно выгрузить на сервера asciinema.org командой asciinema upload [имя-файла]
===
Название: iotop
Описание: Инструмент, предназначенный для мониторинга использования дискового ввод-вывода IRL
Синтаксис: iotop [опции]
Пример: sudo iotop -o (Процессы, фактически выполняющие ввод-вывод)

===
Название: alias  
Описание: Задание псевдонимов командам
Синтаксис: alias [имя]="значение" alias [имя]="команда аргумент1 аргумент2" alias [имя]="/путь/исполняемый файл"
Пример: alias c='clear'
Примечание: Для удаления созданного alias использовать $ unalias [имя]
